# **Instructions**

---

### **Clone this repository**

### **Install all the project's dependencies**

```shell
npm install
```

### **Compile contracts**

```shell
npx hardhat compile
```

### **Run solutions**

```shell
npx hardhat run solutions/challenge*.js
```
