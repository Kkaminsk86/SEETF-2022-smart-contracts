# **Instructions**

---

### **1. Clone this repository**

### **2. Install all the project's dependencies**

```shell
npm install
```

### **3. Compile contracts**

```shell
npx hardhat compile
```

### **4. Run solutions**

```shell
npx hardhat run solutions/challenge*.js
```
